Tools Used: Power BI Desktop, Power Query, DAX

Project Overview:
This project analyzes Twitter engagement data and implements multiple internship tasks using Power BI. The dashboard includes scatter charts, bar charts, line charts, and conditional filtering based on time, dates, and tweet content.

Key Features:

Data cleaning and transformation in Power Query

Custom columns: Hour, Day Number, Character Length, Media Type, App Open Type

Conditional visibility based on time windows

Engagement analysis using replies, likes, retweets, clicks

Task-specific filters and documented dataset limitations

Tasks Implemented:

Media views vs media engagements scatter analysis

Click interactions by tweet category

Top tweets by likes and retweets

Monthly engagement trend

Interaction comparison above median

Engagement rate vs app opens

Note:
Some visuals returned no data after applying all required conditions due to dataset constraints. This has been documented in the project report.
